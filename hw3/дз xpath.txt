В данном задании надо написать правильные xpath пути для следующих элементов:
1. Для сайта google.com обращение к элементу описания результата поиска, используя обращение к ссылке в результате поиска(например, запрос "открытие", ссылка "open.ru",
описание результата "Банк Открытие, тел. 8-800-444-4400 (бесплатный звонок по РФ): весь комплекс услуг для частных лиц, корпоративных клиентов, малого и среднего бизнеса.".
Сделать для одного конкретного примера, предполагая, что ссылка нам известна.

//cite[@class='iUh30 Zu0yb tjvcx' and contains(.,'open.ru')] - ссылка

//span[contains(.,'USD')]/ancestor::tr//td[2] - банк покупает

3. Для страницы https://ru.wikipedia.org/wiki/Список_городов_России обращение к списку элементов названий городов, у которых население более 100000 (100 тыс.)
